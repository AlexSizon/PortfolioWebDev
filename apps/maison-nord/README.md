# Maison Nord

> Premium interior design studio demo built with Astro.

Stack: Astro, Tailwind CSS, EmailJS

## Development

```bash
pnpm --filter @repo/maison-nord dev
```

## Build

```bash
pnpm --filter @repo/maison-nord build
pnpm --filter @repo/maison-nord type-check
```

## Environment

Copy values from `.env.example` and set:

- `PUBLIC_EMAILJS_SERVICE_ID`
- `PUBLIC_EMAILJS_TEMPLATE_ID`
- `PUBLIC_EMAILJS_PUBLIC_KEY`
